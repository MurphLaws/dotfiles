# Keycaps KLP Lamé para ZSA Voyager

STL listos para mandar a un servicio de impresión (JLC3DP, PCBWay, etc.).
Config: stem **Choc** (switches del Voyager) + tapa tamaño **MX**, perfiles Saddle.
Recomendado: resina SLA o nylon MJF (el autor recomienda SLA sobre FDM).

Modelos de [braindefender/KLP-Lame-Keycaps](https://github.com/braindefender/KLP-Lame-Keycaps) — dar crédito al autor al redistribuir.

## Cantidades a imprimir (52 piezas = Voyager completo)

| Archivo | Cantidad | Uso |
|---|---|---|
| `Choc_Stem_MX_Size_Normal_Tilted.stl` | 12 | Fila 1 (números) |
| `Choc_Stem_MX_Size_Saddle_Tilted.stl` | 24 | Filas 2 y 4 |
| `Choc_Stem_MX_Size_Saddle.stl` | 10 | Fila 3 (home), excepto índices |
| `Choc_Stem_MX_Size_Saddle_Homing.stl` | 2 | F y J (bajo cada índice) |
| `Choc_Stem_MX_Size_Thumb.stl` | 2 | Tecla interior del pulgar |
| `Choc_Stem_MX_Size_1.5U_Thumb_V.stl` | 2 | Tecla exterior grande del pulgar |

Consejo: pedir 2-3 repuestos de Saddle Tilted y Saddle por si algún stem sale defectuoso.
