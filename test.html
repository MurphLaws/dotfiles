Introduction

This file demonstrates the three primary ways to create collapsible folds inside a Quarto document.

1. Native Code Folding

Below is a Python code chunk that is folded by default.

👉 Click here to reveal the Python code 
import math

radius = 5
area = math.pi * (radius ** 2)
print(f"The area of the circle is {area:.2f}")
The area of the circle is 78.54

2. Collapsible Callout

Quarto callouts can be made collapsible with collapse="true". They start folded and expand on click.

Note📝 Click to expand this note

This content is hidden until you click the title. Callouts support full Markdown inside them, including lists and code:

Tip💡 A folded tip

Use collapse="false" (the default for some types) to have them open initially.

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 2 * np.pi, 200)
y = np.sin(x)

plt.figure(figsize=(6, 3))
plt.plot(x, y, color="#89b4fa")
plt.title("sin(x)")
plt.tight_layout()
plt.show()

5. Foldable Entire Sections

The previous folds only collapse a block. To fold a whole section — its heading and every paragraph, list, code chunk and image under it — wrap the section in a raw <details> element and put the heading text in the <summary>.

📂 Section A — click to expand the entire section
