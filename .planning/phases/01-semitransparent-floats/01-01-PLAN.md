<objective>
Configure Neovim floating windows and menus to render semitransparent — producing a naturally darker panel effect — while preserving 100% editor transparency and borderless aesthetics.

The snacks picker (user's primary menu: Find Files, Grep) must render semitransparent via window-local `winblend` on each picker window (input, list, preview). Global winblend handles generic floats (NormalFloat, Pmenu, which-key). Telescope is already configured.

Purpose: Floats should feel like a darker panel over the Ghostty wallpaper (via blending), not fully opaque or fully transparent. This improves visual hierarchy without sacrificing the transparent editor aesthetic.

Output: Modified `colorscheme.lua` (global winblend/pumblend) and `snacks.lua` (picker window-local winblend).
</objective>
